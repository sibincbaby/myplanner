# AI Discovery Dashboard

Daily automated discovery of interesting AI/agent projects from across the internet.

## How it works

1. **Discovery** — Agent searches forums, social media, communities, GitHub, and launch platforms
2. **Viability screen** — Each find scored on 4 criteria (weekend-buildable, fills-gap, novel, daily-utility)
3. **Planning** — Viable projects get a full Claude-executable implementation plan
4. **Dashboard** — Everything lands here every morning at 7am

## Recent Digests

Browse the **Daily Digests** sidebar to see what was found each day.
Browse **Project Plans** for viable projects with full implementation plans.
