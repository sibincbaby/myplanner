### Why the ❌ rejections

- **twicc** — a read-only web browser for Claude Code + Codex sessions; useful but not daily-critical, and several Claude-Code UIs (claudecodeui, claude-code-cli-ui) are already tracked. Novelty fails.
- **VibeAround** — 461 stars and polished; a launcher/keeper for multiple coding CLIs overlaps existing multi-agent tools. Fills-gap and novel both fail.
- **MARVIN** — a native macOS SwiftUI pair-programming IDE on the Claude Agent SDK. Ambitious and novel, but a full IDE is not a weekend MVP and it's macOS-only.
- **FinPilot** — AI personal-finance assistant with NL queries; the finance-AI slot was filled yesterday by expense-budget-tracker, and generic expense trackers are plentiful. Fills-gap and novel fail.
- **zeph** — yet another persistent-memory agent; overlaps agentmemory and memorycrystal already seen. Not novel, and not weekend-scoped.
